# microconfig
MCP2200 configurator app using GTK3.

Note: Work in progress

## Requirements

Debian/Ubuntu:
```sh
apt install libgtk-3-dev libusb-dev
```

Arch/Manjaro:
```sh
pacman -S gtk3 libusb
```
